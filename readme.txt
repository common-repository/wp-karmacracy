Allows you to insert Karmacracy links into posts or pages and share your posts on Facebook and Twitter. 

== Description ==

Karmacracy is an easy way to know people sharing relevant content around your social networks, it’s a new generation URL shortener. 

This wordpress plugin will allow you to generate and insert short URLs from this service onto a post or page, resumes your Karmacracy profile in a dashboard widget and allows you to share your published posts on Facebook or Twitter.

This plugin doesn’t automatically generate short links, but allows you to insert short links at will from the post editor.

FEATURES:

* Panel for setting the Karmacracy configuration
* Shortcode icon for easy insertion into a post or page
* Dynamically generates and inserts links onto a post or page
* Dashboard widget that resumes your Karmacracy profile
* Option to share your posts on Twitter and Facebook when are published

Comments, questions, feature requests and bug reports are welcome: [http://www.berriart.com/en/wp-karmacracy/](http://www.berriart.com/en/wp-karmacracy/ "WP Karmacracy")

== Installation ==

1. Extract and upload the directory `wp-karmacracy` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add your Karmacracy Third Party Key in the settings panel
1. That's all

REQUIREMENTS

* Wordpress >= v3.0

== Frequently Asked Questions ==

= Is this an official Karmacracy Plugin? =

No. This plugin is not made by the creators of Karmacracy, this plugin has been developed by [Berriart](http://www.berriart.com/ "Berriart") with the [API](http://www.karmacracy.com/sections/api/wellcome-to-karmacracy-api.php "Karmacracy API") that they provide.

= How can I translate this plugin to my language? =

If you would like to translate WP Karmacracy into another language you can use the .POT file which is included with the plugin. You may want to read the WordPress guide [Translating WordPress](http://codex.wordpress.org/Translating_WordPressTranslating "Translating WordPress"). Before creating a translation file, please check the languages in the WP Karmacracy plugin files to see if one already exists.

If you have created a translation for another language, please send me an email or leave a comment on my blog and I will add it into the main plugin and credit you.
